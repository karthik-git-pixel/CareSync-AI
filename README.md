🏥 CareSync AI+
Predictive Geriatric Telecare & Medication Safety System
CareSync AI+ is an advanced healthcare solution designed to tackle the global crisis of Polypharmacy in elderly care. By combining real-time IoT vitals monitoring with a hybrid AI Risk Engine, we transform reactive medical reviews into proactive, life-saving triage.

🚀 Key Features
🧠 MedGuard AI Engine
Our proprietary risk assessment core uses a hybrid logic approach:

Rule-Based DDI Checker: Instant detection of high-risk Drug-Drug Interactions (e.g., Aspirin + Warfarin) using clinical datasets.

ML PIM Detection: A CatBoost-compatible model that identifies Potentially Inappropriate Medications based on geriatric frailty and age.

MRS Scoring: Generates a weighted Medication Risk Score (MRS) in under 34 seconds.

💬 Gemini-Powered Clinical Bot
An integrated health assistant using the Gemini API that:

Provides 24/7 guidance on fever management and medication safety.

Integrates directly with our custom clinical dataset to offer grounded, patient-specific advice.

⚡ System Backbone
FastAPI & Supabase: A high-performance backend with real-time cloud synchronization for patient profiles and histories.

IoT Vitals Simulator: Streams live BP, Heart Rate, and SpO2 data into the dashboard for dynamic risk adjustment.

HL7-FHIR Interoperability: Supports standardized EHR data imports for seamless hospital integration.
